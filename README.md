# Cash-Flow Tracker
Cash-Flow Tracker is a web-based application that helps users manage their salary and expenses efficiently. It allows users to input their total salary, track expenses, and view the remaining balance in real-time using dynamic JavaScript functionality.

## Features
### Core Functionality
- Input total salary
- Add expenses with name and amount
- Display a list of expenses
- Automatically calculate remaining balance
### Input Validation
- Prevents empty fields
- Restricts negative values
### Data Persistence
- Stores data using browser localStorage
- Retains information after page refresh
### Expense Management
- Delete individual expenses
- Updates balance instantly after deletion
### Data Visualization
- Displays a pie chart representing total expenses and remaining balance using Chart.js

## Technologies Used
- HTML
- CSS
- JavaScript (DOM Manipulation)
- Chart.js
- LocalStorage API
